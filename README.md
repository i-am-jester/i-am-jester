# Hi, I'm Jester!

> My code always tells the truth — usually straight to the console as an `Uncaught Error`.

On a serious note, I build independent software, juggling technologies to fit any task. I blur the lines between web and desktop, creating full-cycle products.

---

### 🎭 My Performance:

* **🌐 Web:** Making websites and web apps that don't crash (at least not while you're looking at them).
* **💻 Desktop:** Wrapping ideas into neat interfaces where the "Close" button actually closes the app.
* **🧠 Philosophy:** Code should be clean, design should be minimalist, and bugs should be well-behaved.
